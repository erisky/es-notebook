git clone https://github.com/the-tcpdump-group/libpcap
sudo apt-get install flex
sudo apt-get install bison
./configure
make 

reference:
http://www.tcpdump.org/pcap.html

Using tcpdump:
basic DPR Check

$ sudo tcpdump -r dpr.dump  "port 13579"

Simpified format:
$ sudo tcpdump -q -i eth2 "port 13579"
ex:
